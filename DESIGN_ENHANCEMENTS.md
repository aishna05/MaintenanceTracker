### If styles don't appear:

   - Chrome: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)

   - Press F12
   - Look for CSS loading errors
   - Verify `style.css` is loading

4. **Verify static files**
   - Check `static/style.css` exists
   - Verify file permissions
